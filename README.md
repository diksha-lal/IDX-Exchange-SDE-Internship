# IDX-Exchange-SDE-Internship

#Project Description

This is a full-stack property search web application similar to Zillow/Redfin that allows users to search and filter through real property listings and view detailed property features with included paginated results.

#Technical Stack: 
- Frontend: React (Create React App)
- Backend: Node.js + Express
- Database: MySQL 8 running in Docker
- Testing: Jest + React Testing Library + Supertest
